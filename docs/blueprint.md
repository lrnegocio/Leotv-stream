# **App Name**: Léo Tv & Stream

## Core Features:

- Secure Admin & User Authentication: Implement user login with distinct admin/user paths, enabling administrators to create user accounts with chosen credentials and assign access levels.
- Comprehensive Content Management: Admin panel for adding, editing, and categorizing movies, series, and episodes with support for a wide array of stream URLs (including YouTube, Dailymotion, playcnvs.stream, visiocine.stream) and a real-time stream testing tool. Supports multiple seasons and episodes per series.
- Flexible Subscription & Access Control: Admin functionality to define and assign subscription tiers including free trials, 30-day, custom, and lifetime access, managing user access without explicit pricing.
- High-Performance PWA User Experience: Develop the application as a Progressive Web App (PWA), ensuring it is installable on user devices, offers an app-like experience, and loads content rapidly across various networks.
- Smart Stream Player with Navigation: An adaptive video player that automatically handles diverse stream links, offers 'zapping' (next/previous channel) and fullscreen buttons, and intelligently prompts to open content in an external player if direct embedment is blocked.
- Voice-Activated Universal Search Tool: Integrate a search bar with a voice command tool (microphone icon) on both the user and admin interfaces, allowing users to find channels, movies, or categories by speaking or typing.
- Content Protection & Parental Lock: Enable administrators to mark specific content as restricted. Users will need to input a master password (customizable in admin settings) to access blocked channels, ensuring controlled viewing.

## Style Guidelines:

- Primary color: A vibrant, modern magenta-purple (#DF4CD9) to convey energy and digital sophistication against dark backgrounds.
- Background color: A very dark, subtle purple-tinted shade (#1E161D) for a cinematic and immersive viewing experience.
- Accent color: A bright, almost electric violet (#A869FF) to highlight interactive elements and create strong visual contrast.
- Body and headline font: 'Inter' (sans-serif), chosen for its modern, clean, and highly readable characteristics, suitable for both large headlines and smaller body text in a content-heavy application.
- Use minimalist and contemporary line icons. Prioritize clarity and intuitive recognition for all navigation, playback controls, and administrative actions to maintain a sophisticated aesthetic.
- Employ a clean, modular grid system for content display. Focus on visual hierarchy with generous spacing to emphasize content, and ensure responsiveness across all device sizes for an optimal viewing experience.
- Incorporate subtle and smooth transitions for navigation, content loading, and user interactions. Animations should enhance the user experience without being distracting, adding a polished and modern feel.
